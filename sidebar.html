Using the addon

Type some chinese text into cell A1 such as 你好. In cell A2, enter the formula 

=PINYIN(A1)
. You should see the text nǐhǎo appear.

Similarly, you can use 

=JYUTPING(A1)
for Cantonese/Jyutping.

Multiple rows

In order to apply the PINYIN formula to multiple rows, use the following syntax: 

=PINYIN(A1:A20)
You can also use the following syntax to apply the formula to all rows (will generate Pinyin for all of column A): 
=PINYIN(A:A)
Or the alternative: (will generate Pinyin for all of column A, starting at row 3) 
=PINYIN(A3:A)

Getting service invoked to many times?

This addon makes requests to an external service to generate Pinyin, and Google limits how many such requests can be made. To avoid hitting this limitation, please remove the PINYIN formula from spreadsheets you are not using, and use the batch mode to apply the PINYIN formula to multiple rows.
